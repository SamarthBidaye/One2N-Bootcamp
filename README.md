Project: Student Management REST API

## Features.
    --> This project is a REST API for managing student records.
        - Adding new student to database.
        - Retrieve all student's records.
        - Retrieve a specific student by ID.
        - Update student details
        - Remove or Delete any student if their data not required.

## Tech Stack
        - Node.js
        - Express.js
        - MongoDB (Database)
        - dotenv

## Prerequisities
    --> Make sure you have installed the following
        - Node.js
        - npm
        - MongoDB (Local Installation or MongoDB Atlas)

## How do i run it in my local system ?
    --> Please follow these steps
        - Clone the repo using git commands in the terminal (git clone <repo url>)
        - Run npm install (This installs all the required dependency)
        - Create a .env file and add the required environment variables.
        - npm run start

<The REST API should now be running on your local machine./>
